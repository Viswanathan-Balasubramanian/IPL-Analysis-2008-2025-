# IPL Data Analytics Dashboard (2008–2025) – Power BI
This project is an end-to-end data analytics solution built using Power BI, focused on exploring, analysing, and visualizing Indian Premier League (IPL) data from 2008 to 2025.
It provides detailed insights into:
•	Player performance (batting, bowling, fielding)
•	Team statistics and season-wise trends
•	Year-wise tournament results
•	Orange Cap, Purple Cap, top fours, and top sixes analysis
•	Points table and team standings
•	Championship history and major milestones
The dashboard is designed to help users, analysts, and cricket enthusiasts explore IPL statistics interactively with clean visuals, advanced DAX calculations, and dynamic filtering.
