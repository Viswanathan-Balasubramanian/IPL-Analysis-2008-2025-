# IPL Data Analytics Dashboard (2008–2025) – Power BI
This project is an end-to-end data analytics solution built using Power BI, focused on exploring, analysing, and visualizing Indian Premier League (IPL) data from 2008 to 2025.
It provides detailed insights into:
•	Player performance (batting, bowling, fielding)
•	Team statistics and season-wise trends
•	Year-wise tournament results
•	Orange Cap, Purple Cap, top fours, and top sixes analysis
•	Points table and team standings
•	Championship history and major milestones
The dashboard is designed to help users, analysts, and cricket enthusiasts explore IPL statistics interactively with clean visuals, advanced DAX calculations, and dynamic filtering.

## IPL Season Analysis (2008–2025)
Description
This dashboard provides year-wise insights into every IPL season from 2008 to 2025.
Users can select any season to view champions, top performers, and summary metrics.
Key Features
Season Highlights
•	Champion team
•	Runner-up team
•	Total matches played
•	Total sixes & fours
•	Centuries and half-centuries
•	Unique venues and total seasons
Orange & Purple Cap Leaders
•	Top run-scorer with runs and team
•	Top wicket-taker with wickets and team
•	Most fours and most sixes (with player images)
Points Table
•	Matches played
•	Wins, losses, ties, NRR
•	Total points
•	Dynamic table updates with selected season
Purpose
This dashboard allows users to compare different seasons and understand how teams and players performed historically.

## IPL Team Analysis
Description	
This dashboard focuses on team-level analytics, offering insights into a selected team’s historical performance from 2008–2025.
Key Features	
Team Summary
•	Total matches
•	Wins, losses, no-results
•	Total sixes & fours
•	Seasons participated
Top Performers (Team-wise)
•	Highest run scorer
•	Highest wicket taker
•	Maximum fours (player)
•	Maximum sixes (player)
Season-wise Win/Loss Trend
•	Bar chart showing wins & losses per season
•	Visual comparison of yearly performance
•	Helps identify rising/falling team trends
Purpose
This dashboard helps analyze how each IPL team has evolved over time, identify strengths, and visualize performance consistency.

## IPL Player Analysis
Description	
This dashboard provides a comprehensive 360° profile of individual IPL players.
It includes batting, bowling, and fielding data for any selected player.
Key Features	
Player Profile Section
•	Player photo
•	Full name
•	Batting style
•	Bowling style
Batting & Fielding Statistics
•	Total Innings
•	Total Runs
•	Number of 4s and 6s
•	Batting Average
•	Strike Rate
•	Number of fifties (50s)
•	Number of centuries (100s)
•	Fielding stats: catches, stumping’s, direct hits
Bowling Statistics
•	Total Overs Bowled
•	Total Wickets
•	3-, 4-, and 5-wicket hauls
•	Bowling Average
•	Bowling Strike Rate
•	Economy Rate
•	Balls bowled and runs conceded
Purpose
This dashboard helps understand player consistency, strengths, and overall contribution across seasons in a visually appealing format.
